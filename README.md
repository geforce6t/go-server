# go server
A template for creating a custom web server using golang. 

- Gin : http web framework
- Gorm
- Redis for storing jwt metadata

### How to run??? : 
Install [air](https://github.com/cosmtrek/air) first.

-------------

- `cp .env.example > .env`
fill the fields in the .env file accordingly

- `go get -u -v`
 
 - `air` / `~/bin/air` 
 > just execute air
